# GitHubRepositorySearch
ストアに公開はしていませんが、  
- インクリメンタルサーチ
- ユーザーのログイン認証
- HTTP通信
- JSONエンコード
- WebView  

を勉強するために作成してみました。

![](https://github.com/KikurageChan/GitHubRepositorySearch/wiki/images/icon.png) 

## 機能
GitHubのユーザIDとパスワードを入力してログインをしてGitHubのレポジトリを検索、閲覧することができます。
